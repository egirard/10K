  <action>
Create `src/engine/scoring.ts` implementing these exported functions:

1. **`buildFrequencyMap(dice: Roll): Map<DieValue, number>`** (internal helper)
   - Count occurrences of each die value

2. **`scoreRoll(dice: Roll): ScoreBreakdown`** (exported)
   - If dice.length === 0, return { total: 0, components: [], scoringDiceIndices: [] }
   - Build frequency map
   - If dice.length === 6, check straight first (freq.size === 6 and all counts === 1), then three pairs (freq.size === 3 and all counts === 2)
   - For remaining dice: iterate frequency map. For each value with count >= 3, score as N-of-a-kind using `TRIPLE_SCORES[value] * Math.pow(2, count - 3)`, consume those dice. For remaining 1s and 5s with count < 3 (after consuming triples), score as singles using SINGLE_SCORES.
   - Track which original dice indices contribute to each component via position mapping
   - Return ScoreBreakdown with total, components array, and scoringDiceIndices

3. **`findScoringDice(dice: Roll): number[]`** (exported)
   - Call scoreRoll(dice)
   - Return scoringDiceIndices sorted ascending
   - Empty array means farkle

4. **`scoreDice(selectedDice: Roll): ScoreBreakdown`** (exported)
   - Score the selected dice using scoreRoll
   - Validate that ALL selected dice participate in scoring (scoringDiceIndices.length === selectedDice.length)
   - If any die is "dead weight", return { total: 0, components: [], scoringDiceIndices: [] } to indicate invalid selection

5. **`bestPossibleScore(dice: Roll): ScoreBreakdown`** (exported)
   - For this implementation, scoreRoll already returns the optimal score due to priority ordering
   - Return scoreRoll(dice)

Key implementation details:
- Import DieValue, Roll, ScoreBreakdown, ScoreComponent from './types.js'
- Import SINGLE_SCORES, TRIPLE_SCORES, STRAIGHT_SCORE, THREE_PAIRS_SCORE, DICE_COUNT from './constants.js'
- The N-of-a-kind formula is: `TRIPLE_SCORES[value] * Math.pow(2, count - 3)` for count >= 3
- For scoringDiceIndices: map from frequency-based scoring back to original array positions. When consuming 3 of value X, find the first 3 indices where dice[i] === X.
- Three pairs [1,1,5,5,3,3] MUST score 1000 (checked before individual scoring)
