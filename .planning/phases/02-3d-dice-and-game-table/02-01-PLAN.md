  <action>
Create the 3D game table scene matching the concept images: green felt surface, dark wood raised rim, fixed top-down camera, warm overhead lighting.

1. **Create src/scene/Camera.tsx** -- Fixed perspective camera:
   - Position: approximately `[0, 12, 5]` looking at `[0, 0, 0]` to create the slightly-angled top-down view shown in concept images (not straight down -- there's visible perspective on the dice and rim)
   - Use `<PerspectiveCamera makeDefault>` from drei
   - fov ~45 for portrait mobile framing
   - No orbit controls -- camera is completely fixed (per user decision: "no camera movement")

2. **Create src/scene/GameTable.tsx** -- Table geometry:
   - **Felt surface:** A Box or Plane at y=0, approximately 8 units wide x 10 units deep (portrait aspect). Color: `#2d6e2d` (rich green felt). Use MeshStandardMaterial with roughness ~0.9 (felt is matte) and no metalness.
   - **Wood rim:** Four Box geometries forming a raised border around the felt. Color: `#4a2a0a` (dark brown wood). Height ~0.5 units above felt. Thickness ~0.4 units. Use MeshStandardMaterial with roughness ~0.6, metalness ~0.05.
   - **Rim corners:** Overlap the rim boxes at corners or use slightly extended lengths so there are no gaps.
   - The felt should sit slightly recessed (y = -0.05) relative to the rim tops so dice can roll on the felt and bounce off the rim walls.
   - The table fills the camera view as shown in the concept images -- wood border visible on all sides, felt occupying the center.

3. **Create src/scene/GameScene.tsx** -- Top-level scene:
   ```typescript
   import { Canvas } from '@react-three/fiber';
   import { Physics } from '@react-three/rapier';
   import { GameTable } from './GameTable';
   import { Camera } from './Camera';

   export function GameScene() {
     return (
       <Canvas shadows>
         <Camera />
         <ambientLight intensity={0.4} color="#fff5e6" />
         <directionalLight
           position={[2, 10, 3]}
           intensity={1.2}
           color="#fff5e6"
           castShadow
           shadow-mapSize-width={1024}
           shadow-mapSize-height={1024}
         />
         <Physics gravity={[0, -9.81, 0]}>
           <GameTable />
         </Physics>
       </Canvas>
     );
   }
   ```
   - Warm lighting: use color `#fff5e6` (warm white) for both ambient and directional lights
   - Single directional light from above-right to create subtle shadows on the dice (when added later)
   - Shadow map 1024x1024 (balance quality vs mobile perf)
   - Physics wrapper with standard gravity -- dice will be added inside this in Plan 02

4. **Ensure the table surface has a physics collider** for dice to land on:
   - Wrap the felt surface in `<RigidBody type="fixed">` with a `<CuboidCollider>` matching its dimensions
   - Wrap each rim wall in `<RigidBody type="fixed">` with `<CuboidCollider>` matching wall dimensions
   - This ensures dice will collide with felt and bounce off walls when added in Plan 03
